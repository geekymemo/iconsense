## 1.0.0

- Initial release of IconSense.

## 1.0.1

- Added standalone font-icon scanning and icon hover preview for HTML or PHP files opened outside the project. This standalone scanning is now isolated from the project scan and the Tree-Shaking report.
- Created a standalone Icon Picker for standalone scans.
- Added image hover support for standalone files. Now you can preview images and SVGs in files opened outside the project.
- Added hover preview for internet-sourced images in all files. If the remoteImageInfo setting is true, you can see the image dimensions and file size below the preview.
- Fixed preview issues for Nuxt, Vue, React, and Next.js projects.
- Logo color changed :D
